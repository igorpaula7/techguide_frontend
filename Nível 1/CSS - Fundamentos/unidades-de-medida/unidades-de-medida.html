Unidades de Medida

No CSS, existem as seguintes unidades de medida:

    Fixas:

  • px - pixels
  • cm - centímetros
  • mm - milímetros
  • in - polegadas
  • pt - pontos (1/72 de polegada)
  • pc - paicas (12 pontos)

    • Relativas:

  • em - muda para os elementos filhos de acordo com o tamanho da fonte (font-size) do elemento pai usado em tipografia!
  • rem - root em, funciona como o em mas se baseia no elemento root
  • % - define a porcentagem que ocupará dentro de um elemento.
  • vw - view width, representa o tamanho total de largura da tela do usuário, em porcentagem
  • vh - view height, representa o tamanho total de altura da tela do usuário, em porcentagem
  • vmin - unidade mínima de tela, em porcentagem
  • vmax - unidade máxima de tela, em porcentagem
