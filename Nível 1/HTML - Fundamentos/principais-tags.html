Tags necessárias para um HTML básico

  • html - determina que o código é html
  • head - cabeça do código
  • meta - define metadados do site
  • tittle - define o título do site
  • link - linka com outros documentos
  • body - corpo do código
  • p - parágrafo
  • h1 - título 1, pode ser h2, h3, h4, h5 e h6
  • img - insere uma imagem
  • form - formulários em html
  • button - botão
  • script - insere código js no html
